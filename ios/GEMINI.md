# ios/GEMINI.md

This directory contains the iOS-specific project files for the `roadmindphone` Flutter application.

## Key Configurations:

*   **Bundle Identifier:** The iOS application bundle identifier is `com.example.roadmindphone`.
*   **Deployment Target:** The minimum iOS deployment target is 13.0.
*   **Swift Version:** The project is configured to use Swift 5.0.
*   **Flutter Integration:** The Flutter module is integrated, with `lib/main.dart` as the entry point.

## Building and Running:

To build and run the iOS application, use the standard Flutter commands from the project root:

```bash
flutter run
```

Alternatively, you can open the `ios/Runner.xcworkspace` in Xcode for native development and debugging.
